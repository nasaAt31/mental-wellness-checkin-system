# 🧠 Mental Wellness Check-In System

## 📖 Description
This is a Java-based console application that helps students track their daily mood and mental wellness. It allows users to record mood entries, analyze trends, and receive simple suggestions.

---

## 🚀 Features
- Add daily mood entry
- View mood history
- Calculate average mood
- Mood trend analysis
- Wellness suggestions
- File handling for data storage

---

## 🛠️ Technologies Used
- Java
- OOP Concepts
- File Handling
- ArrayList

---

## ▶️ How to Run

1. Open project in VS Code
2. Compile:
   javac *.java
3. Run:
   java Main

---

## 📂 Output
Data is stored in:
mood_data.txt
